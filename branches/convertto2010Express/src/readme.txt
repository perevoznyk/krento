Krento Source Code "Read Me"
-------------------------------
There are a few important things to know about the Krento source code
release. Think of this as a "Frequently Asked Questions" ... please read it!

* The source code is not supported in any way. 
  
* There is no design document, or other documentation of any kind.

* Krento setup is not included in the source code release.

Prerequisites
-------------
1. Windows XP, Windows Vista, Windows Server 2003/2008, Windows 7 

2. Visual Studio .NET 2008 Professional Edition
   You should have the C++ compiler installed as well, which may not come installed
   by default.
